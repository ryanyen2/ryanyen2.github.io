Memolet: Reifying the Reuse of User-AI Conversational Memories

UIST 2024

Ryan Yen, Jian Zhao

[ pdf] [ DOI] [ video]

Abstract

To support users in recalling and reusing relevant user-AI conversational memories, we introduce Memolet, an interactive object that reifies memory reuse. Users can directly manipulate Memolet to specify which memories to reuse and how to use them. We developed a system demonstrating Memoletâ€™s interaction across various memory reuse stages.
